## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/subscribe-form.git
```

Go to the project directory

```bash
  cd subscribe-form
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
